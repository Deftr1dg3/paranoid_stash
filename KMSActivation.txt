Windows 10, 11 Pro  Key --> W269N-WFGWX-YVC9B-4J6C9-T83GX

Steps:

1. Insert Key into the system:

slmgr /ipk <kmsclientkey>

2. Set KMS server:

slmgr /skms [kms{7,8,9}.msguides.com, etc...] 

3. Activate inserted key 

slmgr /ato

4. Check activation status:

slmgr /dli
